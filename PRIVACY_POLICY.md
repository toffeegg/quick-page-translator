# Privacy Policy

Last updated: April 23, 2026

## Quick Page Translator

Quick Page Translator is a browser extension that helps users translate text on saved websites and inspect translated or original text with a keyboard-assisted highlight action.

## Information the extension stores

The extension stores the following settings in browser sync storage:

- Saved websites and website-specific translation preferences
- Global translation mode and page translation language rules
- Highlight translation preferences
- Imported settings that you choose to restore through the settings page

This information is stored so the extension can remember your preferences across sessions and, depending on your browser account settings, across signed-in browser profiles that support synced extension storage.

## Information sent over the network

When translation is requested, the extension sends text content to the public Google Translate web endpoint used by the project:

- `https://translate.googleapis.com/`

This can include:

- Text detected on pages you chose to auto-translate
- Text you select when using the `Ctrl` + highlight feature

The project does not operate its own translation server. Translation requests are sent directly from the extension to Google Translate.

## Information the project does not collect directly

Quick Page Translator does not include:

- User account registration
- Analytics or telemetry owned by this project
- Advertising trackers
- A project-run backend database for storing user content

The project maintainer does not directly receive or store your translation history through a first-party server operated by this extension.

## Third-party services

Because translations are performed through Google Translate's public web endpoint, Google may process request data according to Google's own policies and infrastructure. You should review Google's privacy terms if you want more information about how that third-party service handles data.

## Your choices

You can control the extension's behavior by:

- Removing saved websites
- Disabling or pausing auto-translation
- Changing import/export usage
- Uninstalling the extension
- Clearing browser extension storage using your browser tools

## Data retention

Settings remain in browser storage until you change, export, clear, or remove them, or until the extension is uninstalled and your browser deletes its stored extension data.
