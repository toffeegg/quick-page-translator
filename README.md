# Quick Page Translator

Quick Page Translator is a Firefox extension that auto-translates saved websites and gives you a fast `Ctrl` + highlight tool for checking translated or original text on demand.

## Current features

- Auto-translates only the websites you save
- Lets you add or remove the current website directly from the popup
- Includes a popup control to pause or resume auto-translation
- Shows a `Refresh page` action in the popup when auto-translation is paused
- Supports rule-based page translation with up to 3 `From -> To` rows
- Supports two page translation modes:
  `Specific languages`: translates only text matching the configured source languages
  `Entire web page`: translates the page into the selected target languages
- Supports a much wider language list than the initial version
- Lets you choose the target language for `Ctrl` + highlight translation
- Can show the original text when you `Ctrl` + highlight already translated page text
- Watches dynamic pages and translates newly added matching text
- Supports import and export of settings, including saved websites

## Popup behavior

- Shows whether the current website is already saved
- Offers `Add website` or `Remove website` depending on the current tab
- Opens the settings page and closes the popup automatically
- Shows the current translation summary for the active website when available

## Settings page

The settings page currently includes:

- Auto-translate toggle
- Page translation mode selector
- Up to 3 page translation rule rows
- Ctrl + highlight target selector
- Original-text-on-highlight toggle for translated pages
- Saved websites manager
- Import / export tools for settings backup and restore

## Files

- `manifest.json` - extension manifest
- `constants.js` - shared language list and default settings
- `background.js` - translation requests and settings normalization
- `content.js` - page scanning, DOM replacement, live translation behavior, and the `Ctrl` + highlight overlay
- `options.html`, `options.css`, `options.js` - settings UI and settings import/export flow
- `popup.html`, `popup.css`, `popup.js` - popup status, site controls, and pause/refresh actions
- `update-github.bat` - one-click batch helper for stage / commit / push

## Load in Firefox

1. Open `about:debugging` in Firefox.
2. Click `This Firefox`.
3. Click `Load Temporary Add-on`.
4. Choose `manifest.json` from this folder.

## Notes

- This extension uses the public Google Translate web endpoint. It is convenient, but unofficial, and may change or be rate-limited.
- Import / export uses a JSON file generated by the extension settings page.
